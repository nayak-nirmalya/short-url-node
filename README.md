# URL Shortner with Node.js, Express.js, JavaScript, MongoDB.

Features:

- Shorten Any URL.
- Get Analytics.

### Prerequisites

**Node version 18.x.x**

### Cloning the Repository

```shell
git clone https://github.com/nayak-nirmalya/short-url-node.git
```

### Install Packages

```shell
npm i
```

### Setup .env File

```js
MONGODB_URL=
PORT=8001
```

### Start the App

```shell
npm run start
```
